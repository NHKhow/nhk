地图绘制练习

世界地图的简单绘制：

library(maps)
map("world", fill = TRUE, col = rainbow(200),
    ylim = c(-60, 90), mar = c(0, 0, 0, 0))
title("世界地图")

中国地图的绘制：

library(maps)
library(mapdata)
map("china")

美国地图的绘制

library(maps)
map("state", fill = TRUE, col = rainbow(209),
    mar = c(0, 0, 2, 0))
title("美国地图")

美国三州地图：

library(maps)
map('state', region = c('new york', 'new jersey', 'penn'),
    fill = TRUE, col = rainbow(3), mar = c(2, 3, 4, 5))
title("纽约、新泽西和滨州地图")
